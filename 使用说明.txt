*必须在excel表格中填写所设置的操作所需的所有参数
*不要将脚本保存为‘样本文件’，再次打开会被覆盖
*脚本可以放在任意文件夹，脚本中用到的资源要与脚本放在同一文件夹
*所有需要识别图片的路径中不能包含中文，包括图片名字，所识别的图片不要命名为纯数字
*不填的参数位置全部写None
*True和False参数首字母大写，在表格中填写此参数要在前面加上'，如   'True   'False

一：找图（找到目标图并将鼠标移动到该位置（悬停））
                参数1：图片名称(必须是png格式图片，如hello.png，参数1填写’hello‘)
	参数2：识别准确度(选填，默认0.96，填写范围0.1~1.0)


二：点图（找到图片并点击该图片）
	参数和找图一样


三：坐标（鼠标点击设置的坐标位置，点击本工具的坐标获取，然后把鼠标移动到目标位置即可获得坐标值）
	参数1：x值
	参数2：y值
	参数3：鼠标按键（填左或者右，选填，默认为左）
	参数4：点击次数（填写数字，选填，默认为1）
	参数5：每次点击间隔时间（选填，默认为0.1）

四：悬停（将鼠标移动到设置的坐标位置，点击本工具的坐标获取，然后把鼠标移动到目标位置即可获得坐标值）
	参数1：x值
	参数2：y값
	
五：按键 （模拟键盘按下某键，只能填写一个按键，字母小写，如CTRL键：ctrl，如A键：a，如下键：down）
	参数1：按键
	参数2：按下次数（选填，默认为1）
	参数3：每次按下间隔时间（选填，默认为0.1）
六：滚轮 （模拟鼠标滚轮滚动）
	参数1：滑动距离（负数向下滚动，正数向上滚动，选填，默认-300）

七：输入 （向输入框输入内容（复制粘贴，不受输入法影响））
	参数1:想要输入的内容（在执行输入之前，确保光标在输入框中）

八：等待 （延时时间）
	参数1：等待时间（如：10）

九：热键（组合按键）
	参数1：按键1（如：ctrl）
	参数2：按键2（如：v，组合成粘贴功能）

十：拖拽（鼠标左键拖拽到指定坐标，需配合悬停先把鼠标移动到起始位置，再执行拖拽）
	参数1：x값
	参数2：y값

十一：按下（按下按键，相当于按下按键不松开）
	参数1：按键（如：a）

十二：释放（松开按键，把按下的按键松开，没有按下也可以执行，虽然这是没有意义的）
	参数1：按键（如：a）

十三：写入（模拟真实输入，受输入法影响，如果输入法没设置好，会出现输入错误）
	参数1：输入内容
	参数2：每次按下按键间隔时间（决定输入速度）

十四：相移 （相对于鼠标现在的位置做出移动）
	参数1：x값（不是绝对坐标값）
	参数2：y값（不是绝对坐标값）

十五：复制（内置热键）
	不需要参数（使用此操作执行：ctrl+a，ctrl+c）

十六：粘贴（内置热键）
	不需要参数（使用此操作执行：ctrl+a，ctrl+v）

十七：代码（执行指定txt文件中的python代码，只能使用本程序所使用到的库文件）
	参数1：文件名称（只需要输入文件名，不要后缀，文件必须是txt文件，代码中不能定义函数）

十八：截屏（图片名称为设置名称+日时分秒，截图保存路径必填）
	参数1：图片保存名称
	参数2：必填，保存位置 如E:/GCRPA/截图，路劲中必须使用/，使用\可能导致路径错误）

十九：区域找图（和找图一样，但是可以限制扫描范围，速度更快，找到目标图并将鼠标移动到该位置（悬停））
                参数1：图片名称(必须是png格式图片，如hello.png，参数1填写’hello‘)
	参数2：识别准确度(选填，默认0.96，填写范围0.1~1.0)
	参数3：检测左侧起始位置（选填，默认为0，点击本工具的坐标获取，把鼠标放在目标左侧，取x값即可）
	参数4：检测上侧侧起始位置（选填，默认为0，鼠标放在目标上方，取y값即可
                            填写0就是全屏检测，缩小范围可以提高图片识别速度）
	参数5：设置检测区域宽度
	参数6：设置检测区域高度
	参数7：灰度匹配（选填，默认为True，设置为 True 可提升找图速度
	            如果对颜色匹配要求较高设置为 False）

二十：区域点图（和点图一样，限制扫描区域，速度更快，找到图片并点击该图片）
	参数和找图一样
	参数8：鼠标按键（填左或者右，选填，默认为左）
	参数9：点击次数（填写数字，选填，默认为1）

ACRPA - 自动화工作流工具
========================

快速开始
--------
1. 运行程序: python run.py
2. 또는 사용 패키징된 ACRPA.exe

주요 기능
--------
- 스크립트 편집기: 자동화 스크립트를 시각적으로 편집
- 실행 제어: 스크립트 실행, 일시 중지, 중지
- 동작 녹화: 마우스, 키보드 동작을 녹화하고 스크립트 생성
- 템플릿 라이브러리: 내장된 자주 사용되는 자동화 템플릿
- AI 생성: AI를 사용하여 스크립트 코드 자동 생성
- 다크 모드: 밝고 어두운 테마 전환 지원
- 타이머 작업: 자동 실행 시간 설정
- 로그 내보내기: 실행 로그 및 스크린샷 저장

스크립트 형식
--------
엑셀 (.xlsx/.xls) 파일 형식 사용, 각 줄이 하나의 명령을 나타냅니다:
- A열: 명령 유형 (이미지 찾기, 이미지 클릭, 키보드 입력 등)
- B-I열: 명령 매개변수

주의: v0.1.14+ 버전은 기본적으로 .xlsx 형식을 사용하지만 여전히 .xls 형식을 읽을 수 있습니다.

일반 명령어
--------
이미지 찾기:     이미지 이름, 정확도(기본값 0.96)
이미지 클릭:     이미지 이름, 정확도(기본값 0.96)
키보드 입력:     키보드 이름, 횟수, 간격
텍스트 입력:     텍스트 내용
대기:     초
좌표:     x, y, 키보드, 횟수

구성 설정
--------
구성 파일: config.json
- dark_mode: 다크 모드 스위치
- retry_max: 실패 재시도 횟수
- api_key: AI API 키
- app_protect: 앱 보호 스위치
